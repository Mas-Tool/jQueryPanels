Grid Panel

The grid panel allows you to layout elements in a tabular structure or rows and columns. It is similar to the HTML table element but it is more flexible and compatible.
If you have experience with WPF/Silverlight grid panel, you will feel home.

Definition options:

You set row-definitions and column-definitions attributes on the grid (container) element.

Fixed - You can set a fixed value in pixels by setting a number.
Auto - Auto definitions will take as much space as it needs, without respecting the space left.
~Auto - Soft auto definitions will take as much space as it needs as long as there is a free space.
Star (*) - Star definitions will fill the space left (after filling the fixed, auto and ~auto definitions). If you define multiple star columns/rows they will split the space equally. You can spread the space differently by specifying a number with a star.

Properties

You set the attributes on the grid children (1st level only) in order to set their position and span.

grid-row - Sets the row of the element (default: 0)
grid-column - Sets the column of the element (default: 0)
grid-row-span - Sets how many rows the element will span over (default: 1)
grid-column-span - Sets how many columns the element will span over (default: 1)

Events:

gp-item-positioned - Notifies when an item was positioned. Arguments: Grid, Item.

Simple grid definition
row-definitions="*,auto" column-definitions="150,*,*,2*"

You can resize the grid

<div row-definitions="*,auto" column-definitions="150,*,*,2*">
    <div grid-column="0">
        <strong>0. </strong><em>grid-column="0"</em>
    </div>
    <div grid-column="1">
        <strong>1. </strong> <em>grid-column="1"</em>
    </div>
    <div grid-column="2">
        <strong>2. </strong> <em>grid-column="2"</em>
    </div>
    <div grid-column="3">
        <strong>3. </strong> <em>grid-column="3"</em>
    </div>
    <div grid-row="1" grid-column-span="2">
        <strong>4. </strong> <em>grid-row="1"<br/>grid-column-span="2"</em>
    </div>
    <div grid-row="1" grid-column="2" grid-column-span="2">
        <strong>5. </strong> <em>grid-row="1"<br/>grid-column="2"<br/>grid-column-span="2"</em>
    </div>
</div>
$(".grid").gridPanel();

Demo 2

This demo shows the use of auto and soft auto definitions.
The middle grid item will respect its container height and will not overflow.
We also have overflow="auto" so content will be scrolled instead of clipped.
row-definitions="auto,~auto,auto"
0. grid-row="0"
1. grid-row="1"
2. grid-row="2" 
<div row-definitions="auto,~auto,auto">
   <div grid-row="0">
       <strong>0. </strong><em>grid-row="0"</em>
   </div>
   <div grid-row="1" class="gridItem-2">
       <strong>1. </strong><em>grid-row="1"</em>
   </div>
   <div grid-row="2" grid-column="0">
       <strong>2. </strong><em>grid-row="2"</em>
   </div>
</div>
$(".grid").gridPanel();
